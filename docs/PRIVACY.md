# Политика конфиденциальности Click

Дата версии: 2026-05-29

## 1. Какие данные обрабатываются

Click не имеет собственного backend и не собирает централизованную базу пользователей. В браузере пользователя могут храниться:

- факт принятия условий;
- дата принятия условий;
- последние созданные ссылки;
- состояние здоровья shortener-провайдеров.

Эти данные сохраняются в `localStorage` конкретного браузера.

## 2. Данные внутри ссылки

Название, описание, задания и финальный URL кодируются внутри самой Click-ссылки. Любой человек, получивший ссылку, потенциально может извлечь эти данные.

## 3. Передача внешним сервисам

Для сокращения URL приложение отправляет создаваемую ссылку стороннему shortener API (`is.gd`, `v.gd`, `TinyURL`, `CleanURI`). На такую передачу распространяются правила соответствующего провайдера.

## 4. Cookies

Click не устанавливает собственные cookies.

## 5. Аналитика

В базовой версии проекта аналитика отсутствует. Если владелец деплоя добавит аналитику, он обязан обновить эту политику.

## 6. Удаление данных

Пользователь может удалить локальные данные через настройки браузера, очистив данные сайта или `localStorage`.
